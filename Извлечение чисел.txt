Для извлечения чисел из текстовой строки в Python можно использовать регулярные выражения. Для применения этого к нескольким столбцам DataFrame Pandas можно
использовать метод apply().

Пример:

``` python
import re
import pandas as pd

# Создание DataFrame с текстовыми данными и числами в строках
data = {'col1': ['abc123', 'def456', 'ghi789'],
        'col2': ['jkl10', 'mno20', 'pqr30']}
df = pd.DataFrame(data)

# Определение функции для извлечения чисел из строк
def extract_numbers(text):
    numbers = re.findall('\d+', text)
    return ','.join(numbers)

# Применение функции к нескольким столбцам DataFrame
df[['col1', 'col2']] = df[['col1', 'col2']].applymap(extract_numbers)

print(df)
```

Результат:

```
     col1 col2
0  123    10
1  456    20
2  789    30
```

В данном примере мы создаем DataFrame с двумя столбцами, содержащими текстовые данные с числами внутри строк. Затем мы определяем функцию extract_numbers(), 
которая использует регулярные выражения для извлечения чисел из строк и объединяет их через запятую.

Далее мы используем метод applymap() для применения этой функции к столбцам col1 и col2 DataFrame. Результат сохраняется в тех же столбцах, перезаписывая 
исходные данные.

В итоге мы получаем DataFrame с извлеченными числами вместо текстовых данных в столбцах col1 и col2.
